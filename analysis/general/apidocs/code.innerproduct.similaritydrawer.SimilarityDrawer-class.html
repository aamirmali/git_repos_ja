Parameters:
  • npzdict (Npzfile/Dictionary) - the NpzFile object that contains the required data. The object is to be a dictionary containing arrays with keyword 'data_data', 'row_sum_data', 'col_sum_data', 'array_sum_data', 'row_sum_row_sum', and 'col_sum_col_sum'.
  • non_cols (numpy array/list/tuple, shape:(*)) - indicates the cols to be ignored. The data in these columns will not be shown and will not be considered in calculations.
  • non_rows (numpy array/list/tuple, shape:(*)) - indicates the rows to be ignored. The data in these rows will not be shown and will not be considered in calculations.
  • non_detectors (numpy array/list, shape:(*,2)) - indicates the detectors to be ignored. The data in these detectors will not be shown and will not be included in any calculations. This parameter is to be an array of detectors, with each detector represented by (col, row).
  • cut_nons (boolean) - if this parameter is set to True, the data in non_rows and non_cols will be removed from the data. If it is set to False, the data will be set to NaN, and will be shown as white spaces in plots.
  • class_limit (str; numpy array/list/tuple, shape:(2)) - The overarching limit for the class. All other class limit parameters, if not specified, will be set to this limit. This parameter can be None, 'full', 'non_auto', 'inliers' or (x, y). None: limit is not set. 'full': limit is set to (min of data_data, max of data_data). 'non_auto': the limit is set to (min of data_data, max of data_data) with the pointwise product between the same detector removed. 'inliers' the limit is set to (min of data_data, max of data_data) with outliers removed. (x, y): the limit is set to (x, y).
  • data_data_limit (numpy array/list/tuple, shape:(2)) - sets the value of self.data_data_limit
  • row_sum_data_limit (numpy array/list/tuple, shape:(2)) - sets the value of self.row_sum_data_limit

plot_single_block(self, index1, index2, function=None, limit='class', colorbar=True, dominance='column')

source code 

Plots a single block of the pointwise product. The block is either a whole column crossed with another whole column or a whole row crossed with another whole row depending on dominance. The column or row to be crossed is specified with index1 and index2.

Parameters:
  • function (function(element, index)) - a function to be applied to each pointwise product.
  • limit (str; numpy array/list/tuple, shape:(*)) - sets the colour limit. If an element is outside of the limit, it will be set to the respective limit.This parameter can be None, 'class', 'non_auto', or (x,y). None: no limit is set. 'class': the limit is set to self.data_data_limit. 'non_auto': the limit is set to (min in the block, max in the block) with the main diagonals removed. (x,y): limit is set to (x,y).
  • colorbar (boolean) - indicates whether or not a colorbar is shown.
  • dominance (str) - indicates whether a block is a whole column crossed with another whole column or a whole row crossed with another whole row. This parameter can be 'column' or 'row'.
  • index2 (int)
  • index1 (int)

plot_data_data(self, function=None, limit='class', colorbar=True, dominance='column')

source code 

Plots all the data as a 2 dimensional array. if the data is an array of shape (m,n,o,p), it is plotted as a shape(m*o, n*p) array. Data[i][j][k][l] is plotted at point [i*(m-1)+j][k*(n-1)+l]. The data is drawn as a 2 dimensional array of 2 dimensional arrays.

Parameters:
  • function (function(element, index)) - a function to be applied to each pointwise product.
  • limit (str; numpy array/list/tuple, shape:(*)) - sets the colour limit. If an element is outside of the limit, it will be set to the respective limit.This parameter can be None, 'class', 'non_auto', or (x,y). None: no limit is set. 'class': the limit is set to self.data_data_limit. 'non_auto': the limit is set to (min in the block, max in the block) with the main diagonals removed. (x,y): limit is set to (x,y).
  • colorbar (boolean) - indicates whether or not a colorbar is shown.
  • dominance (str) - indicates whether a large block is column crossed with column or row crossed with row. This parameter can be 'column' or 'row'. 'column': data_data_col_dom is used for data. Big blocks represent columns crossed with columns 'row': data_data_row_dom is used for data. Big blocks represent rows crossed with rows.

plot_data_data_average(self, function=None, limit='class', colorbar=True, dominance='column')
