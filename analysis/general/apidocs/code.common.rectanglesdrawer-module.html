draw_image(data, title='', xlabel='', ylabel='', function=None, limit=None, colorbar=True, colorbar_label='')

source code 

Draws the image according to data.

Parameters:
  • data (numpy array, 2 dimensional) - the data to be plotted. The entries in the array specifies the colour of a pixel.
  • title (str) - specifies the title of the plot
  • xlabel (str) - specifies the label on the x axis
  • ylabel (str) - specifies the label on the y axis
  • function (function(element, index)) - a function to be applied to each data element before they are plotted.
  • limit (numpy array/list/tuple, shape:(2)) - indicates the lower and upper limits of colour. Any data element outside of this range will be shown at their respective limit.
  • colorbar (boolean) - indicates whether or not a colorbar is shown.
  • colorbar_label (str) - specifies the label on the colorbar.
